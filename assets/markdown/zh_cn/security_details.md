# 安全详情

安全模式用于提升第三方脚本运行时的安全性。

当安全模式开启时，程序会限制部分高风险 API 的访问权限，以降低脚本对本地数据和系统环境造成影响的可能性。

关闭安全模式并不意味着脚本将获得完整系统权限。所有脚本仍然运行在运行时沙箱中，受到程序环境的约束。安全模式仅作为额外的安全保护层，用于进一步限制部分敏感能力。

如果您信任当前脚本或模组，可以根据需要关闭安全模式，以获得更完整的功能支持。
